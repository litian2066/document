## 无参

```sql
create or replace procedure p_hello is
begin
 dbms_output.put_line('hello world'); 
end p_hello;
```

可以使用命令行操作

```sql
SQL> set serveroutput on
SQL> exec p_querynameandsal(7839);
KING_5000

PL/SQL 过程已成功完成。
```

**一定要有`set serveroutput on`这句，开启输出。**

## 带输入参数的存储过程

````sql
create or replace procedure p_querynameandsal(i_empno IN/*IN表示输入*/ emp.empno%TYPE) as
-- 声明变量
v_name emp.ename%TYPE;
v_sal emp.sal%TYPE;
begin
  -- 查询emp表中某个员工的姓名和薪水并赋值给变量
  select ename, sal  INTO v_name, v_sal from emp where empno = i_empno;
 
  dbms_output.put_line(v_name || '_' || v_sal); 
end p_querynameandsal;
````

## 带输出参数的存储过程

【示例】输入员工号查询

```sql
create or replace procedure p_querysal_out(
      i_empno IN/*IN表示输入*/ emp.empno%TYPE, 
      o_sal out emp.sal%TYPE) as

begin

  select sal  INTO o_sal from emp where empno = i_empno;
  
end p_querysal_out;
```

```sql
declare 
  -- Local variables here
 v_sal emp.sal%TYPE;
begin
  -- Test statements here
  p_querysal_out(7839, v_sal);
  dbms_output.put_line(v_sal);
end;
```
